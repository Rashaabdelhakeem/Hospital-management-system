# Hospital-management-system
This project is my first complete *Odoo development project*, built as part of my training.  
The system is designed to manage:  
- 👩‍⚕️ Doctors  
- 🧑‍🤝‍🧑 Patients  
- 📅 Appointments  
- 💳 Billing  

## 🚀 Features
- Manage patient records efficiently  
- Schedule and track doctor appointments  
- Generate and manage billing/invoices  
- User-friendly interface for hospital staff  

## 🛠️ Technologies Used
- *Odoo* (Python Framework)  
- *PostgreSQL* (Database)  
- *XML & QWeb* (Views & Reports)  
- *GitHub* (Version control)  

## 📌 Purpose
This repository is created as a *practical learning project* to apply and demonstrate Odoo development skills.
